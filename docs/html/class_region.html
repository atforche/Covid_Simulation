Region
Constructor for the Region Class. A Region is a container class that wraps around a set of locations. Locations in different regions are subject to different rules and result in different agent behaviors. Each Region has an associated shape that is rendered to the screen. Use a LocationGenerator to generate Locations within the Region.

Parameters
numLocationsthe number of Locations within this Region
